变形车

....

transform_set_chassis_height(height:float)

设置变形车底盘高度。

• height - 高度，范围：[1-7] 厘米


transform_move_speed(direction:int,speed:int)

变形工程车前进/后退运动。

• direction - 方向（0：前进；1：后退）
• speed - 速度，范围：[5-80] 厘米/秒


transform_turn_speed(turn:int,speed:int)

变形工程车左转/右转运动。

• turn - 方向（2：左转；3：右转）
• speed - 速度，范围：[5-280] 度/秒


transform_move_speed_times(direction:int,speed:int,times:int,unit:int)

控制变形车前后以x速度运动x秒后停止。

• direction - 方向（0：前进；1：后退）
• speed - 速度，范围： [5-80] 厘米/秒
• times - 持续范围，范围：[0-200]
• unit - 单位类型（0：表示按秒运动；1：表示按厘米运动）


transform_turn_speed_times(turn:int,speed:int,times:int,unit:int)

控制变形车左右以x速度运动x秒后停止。

• turn - 方向（2：左转；3：右转）
• speed - 速度，范围： [5-280] 度/秒
• times - 持续范围，范围：[0-200]
• unit - 单位类型（0：表示按秒运动；1：表示按度运动）


transform_move_turn(direction:int,speed:int,turn:int,turn_speed:int)

控制变形车左右运动x秒/度后停止。

• direction - 方向（0：前进；1：后退）
• speed - 前进/后退速度,范围： [5-80] 厘米/秒
• turn - 方向（2：左转；3：右转）
• turn_speed - 旋转速度，范围：[5-280] 度/秒


transform_motor_control(lf:int,rf:int,lb:int,rb:int)

控制变形车四个电机转动。

• lf - 左前轮速度，范围： [-360, 360] 转/分
• rf - 右前轮速度，范围： [-360, 360] 转/分
• lb - 左后轮速度，范围： [-360, 360] 转/分
• rb - 右后轮速度，范围： [-360, 360] 转/分


transform_stop()

变形车停止运动。


transform_arm_control(joint:int,position:int, time:int)

设置变形车四个臂角度。

• joint - 臂(1:左前臂；2:左后臂；3:右后臂；4:右前臂)
• position - 角度，范围： [-180,180 ] 度
• time - 时长，范围： [20, 5000]毫秒


transform_adaption_control(option:bool)

开启/关闭自适应，变形车可以根据不同地形调整姿态。

• option - 开关状态 True表示开，False表示关


案例程序

# 变形车
from ugot.ugot import UGOT
import time

got = UGOT()
# 初始化设备
got.initialize('192.168.8.112')

# # 关闭自适应
got.transform_adaption_control(False)

# 调节高度
got.transform_set_chassis_height(1)
time.sleep(1.5)
got.transform_set_chassis_height(7.2)
time.sleep(1.5)
got.transform_set_chassis_height(5)
time.sleep(1.5)

# 前进/后退
got.transform_move_speed(0, 10)
time.sleep(2)
got.transform_move_speed(1, 10)
time.sleep(2)

# 左转/右转
got.transform_turn_speed(2, 40)
time.sleep(2)
got.transform_turn_speed(3, 40)
time.sleep(2)

# 按里程前进/后退
got.transform_move_speed_times(0, 20, 40, 1)
got.transform_move_speed_times(1, 20, 40, 1)

# 按里程左转/右转
got.transform_turn_speed_times(2, 40, 200, 2)
got.transform_turn_speed_times(3, 40, 200, 2)

# 运动同时转弯
got.transform_move_turn(0, 20, 2, 40)
time.sleep(2)
got.transform_move_turn(1, 20, 3, 40)
time.sleep(2)

# 停止运动
got.transform_stop()
time.sleep(1)

# 四个电机运动
got.transform_motor_control(15, 15, 15, 15)
time.sleep(2)

# 停止运动
got.transform_stop()
